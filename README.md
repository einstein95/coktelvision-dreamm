# How to use this .dreammpack with DREAMM
1. Download the latest here: https://github.com/einstein95/coktelvision-dreamm/releases/latest/download/coktelvision.dreammpack
2. In DREAMM 4.0x21 (or later), select **Manage DREAMMPacks...** from the cog menu. 
3. Drag and drop the coktelvision.dreammpack you just downloaded onto the window.

You should now be able to scan for Coktel Vision games on your computer or drag and drop image files of these games onto DREAMM to import them.

Alternatively:

3. Click on **View DREAMMPack Folder**.
4. Copy the coktelvision.dreammpack file to this folder, close and relaunch DREAMM.
